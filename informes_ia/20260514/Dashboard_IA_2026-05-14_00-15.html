Crítico · Framework agentic

Semantic Kernel: prompt injection con camino real a RCE

CVE-2026-25592 CVE-2026-26030 Prioridad crítica

Microsoft explica que identificó un camino vulnerable en Semantic Kernel capaz de convertir una prompt injection en ejecución remota de código a nivel host. En CVE-2026-26030 la explotación exige vector de prompt injection y uso del Search Plugin con In-Memory Vector Store en configuración por defecto.

Mitigación rápida: actualizar, revisar agentes con tool calling, desactivar configuraciones vulnerables, añadir aprobaciones explícitas y telemetría sobre ejecución de herramientas.

Mitigación rápida: bloquear versión afectada, auditar entornos instalados en la ventana de exposición, revisar lockfiles, contenedores y caches de CI/CD.
